### Блок A (HTML/CSS)
* **task-1:** Профиль пользователя (структура и стилизация).

## 🛠 Инструкция по запуску React-проекта

Проект реализован на **Vite**. Для запуска выполните следующие шаги:

1. Перейдите в папку с React-заданиями:
   ```bash
   cd task-b
Установите необходимые зависимости:

Bash
npm install
Запустите сервер для разработки:

Bash
npm run dev
Откройте в браузере локальный адрес (обычно http://localhost:5173).

Краткое описание проделанной работы
Что было сложно:
Самым серьезным вызовом стала первоначальная настройка окружения на ОС Windows. Столкнулся с необходимостью обновления Node.js до актуальной версии (22.x) и устранением конфликтов в терминале при установке зависимостей Vite. Также потребовалось время, чтобы правильно структурировать импорты компонентов, когда проект начал расти.

Что понравилось:
Больше всего понравилось работать с состоянием в React (useState). Было интересно наблюдать, как интерфейс мгновенно реагирует на изменения данных без перезагрузки страницы. Реализация To-Do листа дала хорошее понимание того, как работают современные веб-приложения с хранением данных в браузере.
